# Kotlin starter

Three minimal, executable kotlin applications.

Multi module Gradle Kotlin DSL.

## Building

```
./gradlew build
```

## Running

### Console

```
./gradlew :console:run
```

### Gui ([JavaFX](https://openjfx.io))

```
./gradlew :gui:run
```

### Web ([Ktor](https://ktor.io))

```
./gradlew :web:run
```
